AI博士：办公Agent的核心是通过MCP/API打通企业工具链——飞书、钉钉、企微、邮件、日历、Jira、Git。单个工具的AI功能有限，但Agent可以跨工具协作。

🔌 以飞书/钉钉集成为例

数据源：消息、文档、日历、审批、任务
Agent能力：
• 监听特定群消息 → 自动提取待办 → 创建任务
• 读取审批流 → 检测超时 → 自动催办
• 汇总一周文档更新 → 生成知识变更周报
• 分析日历冲突 → 建议时间调整

Harness要求：严格的数据权限（只看到应该看到的）、操作审计日志、敏感信息脱敏
3

📝 本节要点

✅ 办公Agent = AI + 企业工具链（飞书/钉钉/邮件/日历/Jira）
✅ 核心价值：消灭重复劳动——日报、会议纪要、邮件分类、消息过滤
✅ MCP协议让工具集成标准化，一次开发多平台复用
✅ Harness重点：数据权限、操作审计、敏感信息脱敏
